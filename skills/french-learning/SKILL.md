# French Learning Skill

## Quick Start

**Send daily lesson:**
```
Leçon du jour: [theme]
Grammaire: [topic]
Vocabulaire: 10 mots
Exercice: [practice]
```

**Answer user questions:**
- Explain grammar rules with examples
- Correct sentences and explain errors
- Provide vocabulary with context
- Create personalized exercises

---

## Daily Lesson Structure

### Morning (9:00) — 1 hour

**20 min — Grammar Essential**
- Focus on one grammar point
- Provide rules + examples
- Practice sentences

**20 min — Vocabulary**
- 10-15 words with phrases
- Theme-based (work, daily life, opinions)
- Context: verb + situation

**20 min — Listening/Repetition**
- Short video/dialogue
- Listen → repeat aloud
- Imitate rhythm

### Afternoon (17:00) — 1 hour

**20 min — Speaking**
- Monologue: tell your day
- Explain an opinion
- Guided dialogue (with user)
- **Silence = no progress**

**20 min — Writing**
- 5-10 sentences
- Real themes: work, country, problem, memory
- Correction + reuse

**20 min — Reading + Correction**
- Short text
- Note useful structures
- Natural expressions

## Monthly Progression

### Month 1 (A2 → B1)

**Grammar Order (Priority):**
1. Present (frequent verbs)
2. Passé composé (avoir)
3. Passé composé (être)
4. Imparfait (basic)
5. Futur proche
6. Articles (le/la/un/une)

**Vocabulary Themes:**
- Work (travailler)
- Daily life (vie quotidienne)
- Opinions
- Travel (voyages)
- Problems & solutions

**Goal:** Speak without blocking, fewer serious errors

### Month 2 (B1)

**Grammar:**
1. Futur simple
2. Pronouns (me, le, lui, y, en)
3. Negation
4. Natural questions
5. Connectors (mais, donc, alors)

**Vocabulary Themes:**
- Feelings (sentiments)
- Technology (technologie)
- Culture (culture)
- Health (santé)
- News (actualités)

**Goal:** Connect ideas, explain/compare, speak 2-3 min without pause

### Months 3-5 (B1 → B2)

**Grammar:**
1. Conditionnel
2. Subjonctif (basic)
3. Logical connectors (cependant, pourtant, toutefois)
4. Nuances (déjà, encore, toujours)
5. Advanced structures

**Vocabulary Themes:**
- Politics (politique)
- Environment (environnement)
- Economics (économie)
- Literature (littérature)
- Academic language

**Goal:** Defend opinion, nuance, precise vocabulary, autonomous language

## Grammar Explanations

### Passé Composé (Avoir)

**Structure:**
```
Sujet + avoir (conjugué) + participe passé
```

**Exemples:**
- J'ai mangé une pomme
- Tu as fini ton travail
- Il a pris le train
- Nous avons vu un film
- Vous avez parlé français
- Ils ont travaillé dur

**Exceptions (Vandrertramp):**
- Venir → venu
- Aller → allé
- Naître → né
- Descendre → descendu
- Rentrer → rentré
- Tomber → tombé
- Rester → resté
- Arriver → arrivé
- Monter → monté
- Partir → parti
- Mourir → mort

### Passé Composé (Être)

**Structure:**
```
Sujet + être (conjugué) + participe passé
+ ACCORD avec le sujet
```

**Exemples:**
- Je suis allé(e) au cinéma
- Tu es parti(e) tôt
- Il est né en 1990
- Elle est arrivée hier
- Nous sommes descendus du train
- Vous êtes montés dans l'avion
- Ils sont morts pendant la guerre
- Elles sont rentrées chez elles

### Imparfait (Basic)

**Structure:**
```
Sujet + radical (nous) + -ais, -ais, -ait, -ions, -iez, -aient
```

**Exemples:**
- Je parlais
- Tu finissais
- Il travaillait
- Nous regardions
- Vous étudiiez
- Ils mangeaient

**Uses:**
1. Repeated actions in past
   - Il jouait au tennis quand il était jeune
2. Description in past
   - Il pleuvait, il faisait gris
3. Habit in past
   - J'allais à l'école en bus

### Futur Proche

**Structure:**
```
Aller (conjugué) + infinitif
```

**Exemples:**
- Je vais manger
- Tu vas partir
- Il va travailler
- Nous allons faire du sport
- Vous allez voir ce film
- Ils vont acheter une maison

## Vocabulary Patterns

**Always include:**
- Verb + context
- Example sentence
- Related words
- Common expressions

**Example (Work):**
```
Travailler — to work

Context:
- Je travaille dans une entreprise
- Il a travaillé dur
- Nous allons travailler ensemble

Related words:
- Le travail — work (noun)
- Le travailleur — worker
- La travailleuse — female worker
- L'emploi — job/employment
- Le salaire — salary

Expressions:
- "Chercher du travail" — look for work
- "Gagner sa vie" — earn a living
- "Au travail" — at work
```

## Exercise Templates

### Grammar Drill

**Pattern:**
```
[Grammar rule] + 5 examples
→ Practice: Create 5 sentences using this rule
```

**Example:**
```
Passé composé with "avoir"

Exemples:
1. J'ai mangé une pizza
2. Tu as bu du café
3. Il a lu un livre
4. Nous avons écrit une lettre
5. Elles ont vu ce film

→ Practice: Write 5 sentences using passé composé with "avoir"
```

### Vocabulary Practice

**Pattern:**
```
[Theme] — 10 words
→ Create sentences with each word
→ Use in a short story
```

**Example:**
```
Travail theme:
1. Travailler
2. L'entreprise
3. Le bureau
4. Le collègue
5. Le patron
6. Le salaire
7. L'emploi
8. La carrière
9. La réunion
10. Le projet

→ Practice: Write a paragraph using 5+ words
```

### Speaking Drill

**Pattern:**
```
[Topic] — Speak for 2 minutes
→ Include: [specific grammar/vocab]
→ Record yourself → listen
```

**Example:**
```
Topic: "Your daily routine"
Time: 2 minutes
Include:
- Passé composé (what you did yesterday)
- Futur proche (what you will do tomorrow)
- 5 vocabulary words from "work" theme

→ Speak for 2 minutes, record, listen, note errors
```

## Error Correction

**Process:**
1. Identify the error type (grammar, vocab, spelling)
2. Show the correction
3. Explain the rule
4. Provide practice example

**Example:**
```
User: "Je suis allé à Paris hier"

Error: Wrong auxiliary
Correction: "J'ai allé à Paris hier" (aller uses "avoir" with passé composé)

Rule: Verbs of movement (aller, sortir, entrer, venir, etc.) usually use "avoir" in passé composé, unless they are intransitive.

Practice: Create 3 sentences with "aller" in passé composé
```
